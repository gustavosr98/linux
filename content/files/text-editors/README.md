```bash
Row start   Ctrl + A
Row end     Ctrl + E
```

## Vim 

```bash
vi file.txt

# Exit
:q!

# Save
:wq

# Undo
u

# Search foward
/ <text>
n # next
N # before

# Search backwards
? <text>

# Replace first <this> on line
:s/<this>/<with_this>/

# Replace all <this> on line
:s/<this>/<with_this>/g

# Replace all <this>
:%s/<this>/<with_this>/g
```
